# KRsweeper 💣

OOP 2026 coursework project — Minesweeper game with a planned AI solver, 
built in Python using Tkinter.

## Timeline
- **Feb 10–15** — Initial Minesweeper logic (no classes)
- **Feb 15–17** — Simple UI with Tkinter
- **Mar 10–13** — Rewrote logic into a single class
- **Apr 15** — Branched the code out to a total of 3 classes (GameEntity, Player and KRsweeper)
- **Apr 17** — Cleaned the code up (Lithuanian function names -> English); Added standart minesweeper mine count colors (1-blue, 2-green, 3-red, etc.); Added a timer to track PB's; Save PB's in scores.json with a password so 2 people with same usernames wouldn't have their score overwritten (**DO NOT USE REAL PASSWORDS, THEY ARE SAVED AS TEXT IN THE JSON FILE**)

## Planned features
- AI solver
- Difficulty presets (Easy / Medium / Hard)
- Timer & scoreboard
